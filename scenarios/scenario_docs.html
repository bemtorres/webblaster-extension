Introduction

Welcome to the WebBlaster API Documentation. This guide will help you understand how to destroy DOM elements programmatically.

Note: This is a simulation environment. No actual data will be lost.

Getting Started

To begin using the library, install it via npm:

npm install webblaster-core

Basic Usage

Import the module and initialize the blaster:

import { Blaster } from 'webblaster-core';

const player = new Blaster({
    power: 9000,
    target: 'div'
});

player.fire();

Configuration Options

The Blaster constructor accepts the following options:

  • power: (Number) The destruction force.
  • target: (String) CSS selector for the target.
  • sound: (Boolean) Enable/Disable sound effects.
Warning: High power settings may cause browser lag due to excessive particle generation.

Troubleshooting

If you encounter issues, please check the console for error messages. Common errors include:

  • TargetNotFoundException: The specified selector matched no elements.
  • OutOfAmmoError: You need to reload.
